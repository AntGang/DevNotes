## 图片        

**!**+**[]**+**()**  []中是名字，（）中是图片地址。点图片即可显示地址

网络图片右键审查元素，在elements中找图片地址，写入（）



## 超链接

快捷键：`ctrl+k`

**[]**+**()**  []中是名字，（）中是链接地址。网站上点文字即可跳转

[点击跳转到百度一下](www.baidu.com)



## 列表

**有序列表**

快捷键：`数字 + . + 空格 `或者`Ctrl + Shift + [`

2. B
3. C      

**无序列表**

快捷键：`减号- +  空格`或者`Ctrl + Shift + ]`

- B
- C
